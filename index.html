신용카드 사용 내역 서식 생성기
지방교육행정 업무용 신용카드 지출 증빙 서식을 자동으로 생성합니다. 정보를 입력하면 A4 규격의 출력용 서식을 즉시 미리볼 수 있습니다.
① 서식 유형 선택
  • 물품: 물품 구매 시 사용. 검수일·검수자 입력란 포함
  • 업무추진비: 회의비·접대비 등 사용 시. 사용 인원 입력란 추가
  • 일반: 그 외 일반 카드 사용 내역
② 카드 정보 입력
  • 품의번호: 해당 건의 품의 번호
  • 카드 사용건명: 품의명 또는 사용 목적
  • 카드 사용일자 / 반납일자: 날짜 선택
  • 카드 사용금액: 원 단위로 입력 (숫자만)
  • 카드 사용자: 실제 카드를 사용한 사람 이름
  • 카드번호 끝자리 4개: 사용 카드 식별용
  • 매출전표 업체: 결제한 가맹점명
③ 추가 정보 입력 (유형별)
  • 업무추진비: 사용 인원(명) 추가 입력
  • 물품: 검수일 및 검수자 입력
  • 일반 유형은 추가 입력 항목 없음
④ 서식 생성 및 출력
  • 서식 생성 버튼 클릭 → 우측 미리보기에서 확인
  • 인쇄 버튼 클릭 → 브라우저 인쇄 창 열림
  • 인쇄 설정에서 여백을 "없음"으로 설정하면 더 깔끔하게 출력됨
  • PDF로 저장하려면 프린터를 "PDF로 저장" 선택
📌 입력한 정보는 브라우저 내에서만 처리되며 외부로 전송되지 않습니다.
