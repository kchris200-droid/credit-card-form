# 신용카드 사용 내역 서식 생성기

지방교육행정 업무용 신용카드 지출 증빙 서식을 자동으로 생성하는 도구입니다.  
정보를 입력하면 A4 규격의 출력용 서식을 즉시 생성합니다.

---

## 주요 기능

- 서식 유형 3종 지원: 물품 / 업무추진비 / 일반
- 카드 정보, 사용일자, 금액, 사용자 등 입력 후 서식 즉시 생성
- A4 용지 기준 인쇄 최적화 (브라우저 인쇄 기능 사용)
- 매출전표 부착란 자동 높이 조정
- 물품 서식: 검수 확인란 포함
- 업무추진비 서식: 사용 인원 입력 지원

---

## 사용 방법

1. **서식 유형 선택**: 물품 / 업무추진비 / 일반 중 해당 유형 클릭
2. **카드 정보 입력**
   - 품의번호, 카드 사용건명(품의명)
   - 카드 사용일자 및 반납일자
   - 카드 사용금액, 사용자, 카드번호 끝 4자리
   - 매출전표 업체명
3. **추가 정보 입력** (유형별)
   - 업무추진비: 사용 인원(명) 추가 입력
   - 물품: 검수일 및 검수자 입력
4. **서식 생성** 버튼 클릭 → 우측 미리보기 확인
5. **인쇄** 버튼 클릭 → 브라우저 인쇄 창에서 PDF 저장 또는 출력
   - 인쇄 설정에서 여백을 "없음"으로 설정하면 더 깔끔하게 출력됩니다

---

## 기술 스택

순수 HTML/CSS/JavaScript 단일 파일로 구성되어 있으며, 별도 서버나 설치 없이 브라우저에서 바로 실행됩니다.

입력한 정보는 브라우저 내에서만 처리되며 외부 서버로 전송되지 않습니다.
